# A-Priory
A-Priory
Implementation of A-Priory algorithm using following transactions:
   
    ['I1', 'I2', 'I3'],  # T1
    ['I2', 'I3'],        # T2
    ['I1', 'I2', 'I3'],  # T3
    ['I1', 'I3'],        # T4
    ['I1', 'I2', 'I3'],  # T5
    ['I1', 'I2', 'I3'],   # T6
    ['I1', 'I2', 'I3', 'I4'],   # T7
    ['I1', 'I2', 'I4']   # T8
